# configs
My Personal dots for dwm , st and dmenu
## dependencies
feh   
picom   
a way to start xinitrc  
dunst  
fish (no config files!)   
cbonsia (with -s flag)   
cmatrix   
    
## how to complie
### 1.clone the repo

### 2.Suckless software
  2.1. enter every suckless apps dir
  2.2. sudo make clean install  , to install it
### 3.Picom
  3.2 move the picom.conf to c/etc/xdg/
### 4.Wallpaper
  4.1 get the wallpaper from https://wallhaven.cc/w/exj2yl
  4.2 Download it to ~/Pictures/
  4.3 Name it archbtw
### 5.Get it All working!
  5.1 Copy the .xinitrc to your .xinitrc
### 6.Baam
